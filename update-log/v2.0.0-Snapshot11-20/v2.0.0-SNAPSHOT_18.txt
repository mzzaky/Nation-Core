====================================================
  CALIPHATE — NEW GOVERNMENT TYPE
====================================================
Implementasi jenis pemerintahan baru: CALIPHATE (Khilafah).
Caliph memimpin selamanya tanpa kabinet/menteri; Shura Council (3) dan State
Scholars (5) hanya badan penasihat tanpa wewenang order; satu-satunya alat
governance Caliph adalah executive order standar (cooldown tetap berlaku —
Caliph TIDAK bebas cooldown layaknya King).

[NEW GOVERNMENT TYPE — CALIPHATE]
- GovernmentType.java:
  → Penambahan enum CALIPHATE (icon TURTLE_HELMET, warna dark green).
  → getLeaderTitle() = "Caliph", getCitizenTitle() = "Citizen".
  → fromString() menerima alias "caliphate", "khilafah", "kalifah", "caliph".
- ExecutiveOrder.java:
  → NationType ditambah varian CALIPHATE ("Caliphate" / "Caliph").
  → createForNation() routing untuk CALIPHATE → CaliphateExecutiveOrder.
- Nation.java:
  → Field baru caliphateGovernment + getter/setter.
  → initCaliphateGovernment() otomatis dipanggil saat type == CALIPHATE:
    founder ditahbiskan sebagai Caliph pertama.

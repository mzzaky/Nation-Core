====================================================
  NATION CORE - UPDATE LOG v2.0.0-SNAPSHOT#27
====================================================
Revamped administrative commands by introducing a flexible treasury control system, adding target player invoice generation/settlement options, transitioning leader, announcement, and NPC setups to use nation name parameters instead of IDs, and completely removing the obsolete admin action and reset command suites along with their associated backend logic.

[ADMIN TREASURY CONTROLS REVAMP]
- NationCommand.java (id.nationcore.commands):
  → Replaced `/dc admin addtreasury <nation_id> <amount>` with `/dc admin treasury give <nation_name> <amount>` to add funds to a nation's treasury using its name.
  → Added `/dc admin treasury take <nation_name> <amount>` to withdraw funds from a nation's treasury using its name.
  → Added `/dc admin treasury set <nation_name> <amount>` to set a nation's treasury balance to a specific amount using its name.
  → Implemented full tab-completion supporting the new action types (`give`, `take`, `set`) and dynamic nation name retrieval.

[ADMIN INVOICE MANAGEMENT COMMANDS]
- NationCommand.java (id.nationcore.commands):
  → Added `/dc admin invoice add <player_name>` to manually generate and assign a new tax invoice to a target player if they belong to a nation.
  → Added `/dc admin invoice remove <player_name>` to settle all active invoices (including overdue or national debt) for a player and route the funds to their nation's treasury.
  → Implemented full tab-completion supporting `add` and `remove` actions followed by online player names.
- TaxManager.java (id.nationcore.managers):
  → Changed `applyPayment` from private to public to allow manual admin invoice settlements to process payments correctly.

[ADMIN LEADER, ANNOUNCEMENT & NPC COMMAND ADJUSTMENTS]
- NationCommand.java (id.nationcore.commands):
  → Modified `/dc admin set president <player_name>` to `/dc admin set leader <nation_name> <player_name>` to manually appoint a leader for any nation type by its name with full tab-completion.
  → Adjusted `/dc admin set announcement <nation_id> <custom_message>` to `/dc admin set announcement <nation_name> <custom_message>` with full tab-completion.
  → Overhauled the `/dc admin npc <invite|kick|role|list>` subcommands to take `<nation_name>` instead of `<nation_id>` for better readability.
  → Upgraded tab-completion for NPC subcommands to dynamically resolve multi-word nation names, showing lists of fake members for kick/role actions and available roles ("CITIZEN"/"OFFICER") at correct arguments.

[OBSOLETE COMMAND REMOVALS]
- NationCommand.java (id.nationcore.commands):
  → Completely removed the `/dc admin action` command suite and all its subcommands.
  → Completely removed the `/dc admin reset` command suite and its subcommands (`government`, `election`, `treasury`, `all`).
- DataManager.java (id.nationcore.managers):
  → Deleted the unused backend reset helper methods: `resetGovernment()`, `resetElection()`, and `resetTreasury()`.

[LANGUAGE CONFIGURATION CLEANUP]
- language.yml (src/main/resources):
  → Removed unused keys: `general.data_reset`, `general.all_data_reset`, `admin.help_addtreasury`, `admin.help_reset`, `admin.addtreasury_usage`, `admin.addtreasury_success`, `admin.addtreasury_invalid`, `admin.reset_usage`, and `admin.reset_invalid`.
  → Updated help descriptions to reflect the new commands.
